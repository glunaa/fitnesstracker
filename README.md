# 🏋️‍♂️ Fitness Tracker App

A console based tracking app built with Python, designed to help users monitor their health metrics including BMI, daily protein intake, and Basal Metabolic Rate (BMR).

## 💡 Overview

This app provides quick and accurate fitness calculations to support users in reaching their health and wellness goals. Whether you’re tracking your weight, planning your macros, or estimating your energy needs, this tool has you covered.

## ⚙️ Features

• 🧮 BMI Calculator – Computes Body Mass Index based on height and weight
• 🍗 Protein Intake Calculator – Estimates daily protein needs based on activity level and body weight
• 🔥 Basal Metabolic Rate (BMR) – Calculates resting energy expenditure using Mifflin-St Jeor or Harris-Benedict formulas
• 💾 Optional localStorage support for saving user inputs

## 🚀 Getting Started

# Clone the repository
git clone https://github.com/glunaa/fitnesstracker.git

# Navigate into the project folder
cd fitnesstracker

# Install dependencies
npm install

# Start the development server
npm run dev
